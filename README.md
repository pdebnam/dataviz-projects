# dataviz-projects





A collection of small, exploratory data visualization projects built in Tableau.

These are passion projects created to explore interesting questions, test design

ideas, and practice data storytelling.



\## Tools

\- Tableau

\- Excel / CSV

\- Public datasets (sources cited per project)



\## Projects



\### 🏀 March Madness: How Long Do Perfect Brackets Survive?

\*\*Question:\*\* How long do “perfect” NCAA Tournament brackets typically last?



\*\*Highlights:\*\*

\- Early‑round chaos dominates

\- Several years busted before the Round of 64 concluded

\- 2020 excluded due to tournament cancellation
